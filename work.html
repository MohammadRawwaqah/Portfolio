CMS and Shopping Cart

Content Managment System and Shopping cart, it is a personal project and the idea to build this application is to employ what i learn (Front-end and Back-end) in this web application.

This application is an online shopping cart aimed to provide convenience to customers and adding the feature of displaying products with the highest accuracy and largest size to allow the customer to properly preview them before purchasing. it is more than just only a display of products, but it allows the admin to add and modify all of the products, pages, pages content and delete them or arrange them as desiered, all these features are granted to the admins role. when we talk about roles, this app has three roles (Guest, Customer, Admin)

All operations performed on this application reach the admin via e-mail with all information related to (quantity, time, customer name ..etc).

  • ASP.NET Web API
  • Entity Framework
